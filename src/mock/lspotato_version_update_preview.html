🧪 Quick Testing Guide

Step 1: Enable Mock Mode in utils.py

# Uncomment these lines in check_for_updates(): from .mock_update_data import get_mock_update_data return get_mock_update_data()

Step 2: Change scenario in mock_update_data.py

ACTIVE_SCENARIO = "SCENARIO_BOTH" # Change this line

Step 3: Available scenarios:

  • SCENARIO_NO_UPDATE - No updates
  • SCENARIO_PATCH_ONLY - Only v1.0.20
  • SCENARIO_MINOR_ONLY - Only v1.1.1
  • SCENARIO_BOTH - Both options ⭐
  • SCENARIO_MAJOR - Major update v2.0.0

Step 4: Reload Blender addon and click "Check for Updates"
